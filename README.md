# 민혁이의 보물창고 - 포트폴리오

React + TypeScript + Vite로 제작된 개인 포트폴리오 웹사이트입니다.

## 🚀 배포 주소
- **프로덕션**: [여기에 Render 배포 주소 추가 예정]

## 🛠️ 기술 스택
- React 18
- TypeScript  
- Vite
- Tailwind CSS
- Framer Motion

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

AI와 웹 풀스택 개발에 열정을 가진 개발자 조민혁의 포트폴리오입니다.
